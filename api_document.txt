API Documentation

http://localhost:3000/api/v1

1. Đăng nhập

    URL: /login
    Phương thức: POST
    request body:

    json

    {
      "username": "string",
      "password": "string"
    }

2. Lấy thông tin khoa

    URL: /get_khoa/:khoaId
    Phương thức: GET

3. Lấy thông tin sinh viên

    URL: /get_sinh_vien/:sinhVienId
    Phương thức: GET

4. Lấy thông tin giảng viên

    URL: /get-giang-vien/:giangVienId
    Phương thức: GET

5. Lấy thông tin bộ môn

    URL: /bo-mon/:boMonId
    Phương thức: GET

6. Lấy thông tin môn học

    URL: /mon-hoc/:monHocId
    Phương thức: GET

7. Lấy thông tin phòng học

    URL: /phong-hoc/:phongHocId
    Phương thức: GET

8. Lấy thông tin tòa nhà

    URL: /toa-nha/:toaNhaId
    Phương thức: GET
